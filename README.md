pxEmoji is a lightweight Minecraft plugin that adds emoji (Unicode symbols) to chat tab completion and ensures correct rendering of these symbols on signs

---

## Features

- Adds emoji symbols to chat tab completion
- Allows players to quickly insert emojis using TabComplete
- Ensures correct formatting and rendering of emojis on signs

---

## Permissions

| Permission | Description |
|-----------|-------------|
| `pxemoji.reload` | Allows usage of `/pxemoji reload` |
